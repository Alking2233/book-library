# 📚 Book Library | مكتبة الكتب

مشروع متكامل لمكتبة كتب إلكترونية يتيح للمستخدمين تصفح الكتب، البحث عنها، وعرض تفاصيلها.  
يحتوي المشروع على **واجهة أمامية** مبنية بـ React، و**واجهة خلفية** مبنية بـ Strapi لإدارة المحتوى.

---

## 🚀 التقنيات المستخدمة | Tech Stack

### 🎨 Frontend
- ⚛️ React JS
- 💛 JavaScript
- 🎨 CSS / Tailwind (حسب المشروع)
- 🔗 Axios (للتواصل مع API)
- 🧭 React Router

### 🛠️ Backend
- 🟣 Strapi (Headless CMS)
- 🗄️ SQLite

## ✨ المميزات | Features

- 📖 عرض قائمة الكتب
- 🔍 البحث عن الكتب
- 📂 تصنيف الكتب حسب الفئات
- 📝 عرض تفاصيل كل كتاب
- 🛒 إضافة الكتب إلى السلة 
- 🧑‍💼 لوحة تحكم لإدارة الكتب من Strapi
- 📱 تصميم متجاوب يعمل على جميع الأجهزة
